# SpringBoot

**Фреймворк** - это каркас или структура проекта. Подключая фреймворк в проект, он определяет структуру этого проекта, 
разработчику требуется только писать код в определённых фрейморком классах, например, контроллеры, сервисы и т.п.

**Spring Framework** - наиболее популярный фреймворе для веб разработки на Java. Имеет в себе решения на множество 
популярных задач, например, доступ к БД.

## Spring Framework

Наиболее популярный и полезный функционал:

1. Контекст приложения (_Application Context_) & Внедрение зависимостей (_Dependency injection_)

2. Удобный доступ к БД (_Spring Data JPA_)

3. Шаблон "Модель - Вид - Контроллер" (_MVC_)
