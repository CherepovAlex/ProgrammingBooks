# Политики вытеснения (Cache replacement policies)

**Вытеснение давно используемого (Least recently used, LRU)**
- в первую очередь вытесняется элемент неиспользованный дольше всех. 

**Вытеснение наименее часто используемого (Least-Frequently Used, LFU)**
- в первую очередь вытесняется элемень, обращения к которому происходит реже всего.
